# Bones HTML

## Compiling assets

We are using Gulp JS on this project. To compile assets there are several simple steps:

```
npm install
```

and then you have two options:

run watch function:

```
gulp watch
```

compile assets and exit:

```
gulp dev
```

## Tool versions used for successfull compile

- npm 6.9.0
- node v12.5.0
- python 2.7.9
